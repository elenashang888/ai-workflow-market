## 🎯 功能清单

### 核心功能

1. **每日自动选题**
   - 监控小红书热门话题
   - 分析竞品爆款内容
   - 智能推荐今日选题
   - 推送至微信/邮件

2. **AI文案生成**
   - 基于选题自动生成标题
   - 生成3个版本正文（短/中/长）
   - 自动添加emoji和话题标签
   - 优化关键词密度

3. **自动配图生成**
   - Midjourney自动生成封面图
   - 多风格可选（ins风/极简/复古等）
   - 自动添加文字标题
   - 生成3张备选

4. **定时发布**
   - 设置最佳发布时间（早8点/午12点/晚8点）
   - 自动发布到小红书
   - 发布前人工审核开关
   - 发布失败自动重试

5. **评论自动回复**
   - AI识别评论意图
   - 常见问题自动回复
   - 好评自动感谢
   - 负面评论预警通知

## 📖 详细搭建步骤

### 步骤1：创建Make.com场景

1. 登录 Make.com
2. 点击 "Create a new scenario"
3. 搜索并添加 "Schedule" 模块
4. 设置触发时间：每天 7:00 AM

### 步骤2：添加热门话题监控

1. 添加 "HTTP" 模块
2. 配置小红书热门榜单API
3. 添加 "JSON" 解析模块
4. 提取热门话题数据

### 步骤3：添加ChatGPT文案生成

1. 添加 "OpenAI" 模块
2. 选择 "Create a Completion"
3. 配置Prompt：

```
你是一位专业的小红书文案写手。

请基于以下热门话题，创作一篇小红书笔记：
话题：{{热门话题}}

要求：
1. 标题要吸引人，使用数字或悬念
2. 正文分3-5段，每段不超过100字
3. 添加适当的emoji
4. 包含3-5个相关话题标签
5. 语气亲切自然，像朋友分享

请输出：
- 标题（3个备选）
- 正文内容
- 话题标签
```

### 步骤4：添加Midjourney配图生成

1. 添加 "HTTP" 模块
2. 调用Midjourney API
3. 配置Prompt：

```
A beautiful lifestyle photo for Xiaohongshu post, 
{{风格描述}}, soft lighting, pastel colors,
minimalist composition, high quality, 4k,
ar 3:4 --v 6.0
```

### 步骤5：添加发布模块

1. 添加 "HTTP" 模块
2. 配置小红书发布API
3. 设置发布参数
4. 添加错误处理

### 步骤6：添加评论监控

1. 添加 "Webhook" 模块
2. 配置评论监听
3. 添加 "OpenAI" 模块分析评论
4. 设置自动回复规则

### 步骤7：测试运行

1. 点击 "Run once" 测试
2. 检查每个模块输出
3. 调试错误
4. 启用自动运行
