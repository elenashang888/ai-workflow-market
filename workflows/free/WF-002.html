📋 工作流概述

这个工作流将帮你实现公众号的自动化运营，从热点监控到文章发布，全流程自动化。特别适合内容创作者、企业公众号运营者。

核心功能

  • 热点监控 - 自动追踪行业热点和 trending 话题
  • 大纲生成 - AI自动生成文章大纲和结构
  • 文章撰写 - 基于大纲自动生成完整文章
  • 封面制作 - 自动生成公众号封面图
  • 定时发布 - 选择最佳时间自动发布
💡 适合人群

公众号运营者、内容创作者、企业新媒体团队、自媒体人

🛠️ 准备工作

在开始搭建之前，你需要准备以下账号：

必需账号

  • ✓ Make.com - https://www.make.com（免费版）
  • ✓ OpenAI API - https://platform.openai.com
  • ✓ 公众号 - 已认证的公众号
  • ✓ Midjourney - 用于生成封面（可选）
⚠️ 重要提醒

公众号对自动化发布有限制，建议使用"人工审核模式"，即工作流生成内容后发送给你确认，确认后再手动发布。

🔧 搭建步骤

按照以下步骤，约35分钟即可完成搭建：

1
创建Make.com场景
  1. 登录 Make.com
  2. 点击 "Create a new scenario"
  3. 命名为 "公众号文章自动化"
2
设置定时触发器
  1. 添加 "Schedule" → "Run on a schedule"
  2. 设置每天早上 8:00 运行
💡 最佳发布时间

早高峰：7:00-9:00 | 午高峰：12:00-13:00 | 晚高峰：20:00-22:00

3
配置热点监控

使用 RSS 或 API 监控热点：

  1. 添加 "RSS" → "Retrieve RSS feed items"
  2. 输入热点源（如：微博热搜、百度热搜）
  3. 设置获取最新的5条热点
4
配置ChatGPT文章生成

添加 OpenAI 模块，配置Prompt：

你是一位专业的公众号写手，擅长创作爆款文章。 请基于以下热点，创作一篇公众号文章： 热点：{{热点标题}} 要求： 1. 标题要有吸引力，使用数字、悬念或情感共鸣 2. 正文1500-2000字，分5-7个小节 3. 每节有小标题，结构清晰 4. 语言通俗易懂，有深度但不晦涩 5. 结尾有金句总结和互动引导 请输出： 标题：[文章标题] 正文：[文章正文]
5
设置通知模块

添加邮件或微信通知，接收生成的文章：

  1. 添加 "Email" → "Send an email"
  2. 配置收件人：你的邮箱
  3. 邮件内容：文章标题 + 正文 + 建议封面图关键词
6
测试和启用
  1. 点击 "Run once" 测试运行
  2. 检查收到的邮件内容
  3. 调整Prompt优化内容质量
  4. 开启 "Scheduling" 启用自动运行

💡 使用技巧

  • 建立选题库：在Google Sheets中积累热点关键词，让AI从中选择
  • 风格训练：在Prompt中加入你过往爆款文章作为示例
  • 人工润色：AI生成后人工优化，加入个人观点和经验
  • 封面制作：使用Canva或Midjourney根据关键词生成封面

❓ 常见问题

Q: 生成的文章会被判定为抄袭吗？

A: AI生成的内容是原创的，但建议进行人工润色，加入个人经验和观点。

Q: 可以发布到其他平台吗？

A: 可以！修改Prompt适配不同平台风格，如知乎、头条、百家号等。

Q: 为什么我的阅读量不高？

A: 自动化工具只是提升效率，内容质量和运营策略才是关键。建议多研究爆款文章。
