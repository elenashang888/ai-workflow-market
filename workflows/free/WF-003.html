📋 工作流概述

这个工作流帮你批量生成短视频脚本，包括分镜设计、口播稿、标签建议。适合抖音、视频号、小红书等平台。输入一个主题，自动生成10条脚本！

核心功能

  • 脚本生成 - 批量生成10+条视频脚本
  • 分镜设计 - 每个脚本包含详细分镜
  • 口播稿 - 生成自然流畅的口播文案
  • 标签建议 - 自动推荐热门标签
  • 发布排期 - 自动生成发布时间表
💡 适合人群

短视频创作者、MCN机构、自媒体人、企业短视频运营

🛠️ 准备工作

  • ✓ Make.com账号 - https://www.make.com
  • ✓ OpenAI API - https://platform.openai.com
  • ✓ Google Sheets - 用于存储生成的脚本

🔧 搭建步骤

1
创建Google Sheets表格
  1. 创建新表格，命名为"短视频脚本库"
  2. 设置表头：主题 | 脚本标题 | 口播稿 | 分镜 | 标签 | 状态
  3. 分享表格，获取可编辑链接
2
创建Make.com场景
  1. 创建新场景，命名为"短视频脚本生成"
  2. 添加触发器：Webhook（用于接收主题）
3
配置ChatGPT脚本生成

添加OpenAI模块，配置Prompt：

你是一位专业的短视频编剧，擅长创作爆款脚本。 请基于主题"{{主题}}"，创作10个短视频脚本。 每个脚本包含： 1. 标题（15字以内，有吸引力） 2. 口播稿（30-60秒，口语化，自然流畅） 3. 分镜设计（3-5个镜头，描述画面和时长） 4. 推荐标签（5个相关标签） 风格要求： - 开头3秒要有"钩子" - 内容要有价值或情感共鸣 - 结尾有互动引导 - 适合抖音/视频号平台 请以JSON格式输出，方便解析。
4
配置Google Sheets写入
  1. 添加 "Google Sheets" → "Add Row"
  2. 连接你的Google账号
  3. 选择"短视频脚本库"表格
  4. 映射字段：主题、标题、口播稿、分镜、标签
5
测试和使用
  1. 复制Webhook URL
  2. 在浏览器访问：URL?主题=你的主题
  3. 检查Google Sheets是否收到数据
  4. 启用场景

💡 使用技巧

  • 批量生成：一次输入多个主题，用逗号分隔
  • 风格调整：在Prompt中加入风格要求，如"搞笑风格"、"知识科普风格"
  • 爆款复刻：输入爆款视频链接，让AI分析并生成类似脚本
  • A/B测试：同一主题生成多个版本，测试哪个表现更好
